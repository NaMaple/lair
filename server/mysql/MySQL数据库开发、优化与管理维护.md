| 类型 | 字节  | 范围(有符号) | 范围(无符号) | 
| :---: | :---: | :---: | :---: |
| tinyint | 1 | -128~127| 0~255 |
| smallint | 2 | -32768~32767 | 0~65535 |
| mediumint | 3 |||
| int | 4 |||
| bigint | 8 |||
